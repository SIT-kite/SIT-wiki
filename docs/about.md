# 关于上应 Wiki

## MkDocs

本项目使用了 [MkDocs](https://www.mkdocs.org/)，本项目和许多其他 MkDocs 项目使用了
[Material for MkDocs](https://squidfunk.github.io/mkdocs-material/)。
